# 五子棋小游戏

一个支持双人对战和人机对战的五子棋游戏。

## 功能特性

- **双人对战**：两名玩家轮流下棋
- **人机对战**：玩家vs AI
- **三种AI难度**：
  - 简单：随机落子 + 基础防守
  - 中等：位置评估 + 贪心算法
  - 困难：Minimax + Alpha-Beta 剪枝

## 如何运行

直接在浏览器中打开 `index.html` 即可开始游戏。

## 游戏说明

1. 选择游戏模式（双人对战 / 人机对战）
2. 人机模式下可选择AI难度
3. 黑方先行
4. 率先连成五子者获胜

## 技术栈

- HTML5 Canvas
- JavaScript (原生)
- CSS3
